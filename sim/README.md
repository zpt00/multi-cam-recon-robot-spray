# Simulation — FANUC Roboguide Paint

This directory contains the FANUC Roboguide paint simulation workspace used to
validate spray trajectories before deploying to the physical robot.

## Contents

- `paint/` — Main Roboguide simulation project (`.ptw` workspace)
  - `layout/` — Workcell layouts with robot, workpiece, and camera rig
  - `object/` — CAD models of workpieces
  - `ProductionScenarios/` — Simulation run configurations
  - `example_savepoints/` — Representative saved robot programs

## Usage

1. Open `paint/实验室镜像备份仿真.ptw` in FANUC Roboguide
2. Load the desired production scenario
3. Run simulation to validate:
   - Robot reachability for all trajectory points
   - Collision detection between spray gun and workpiece
   - Cycle time estimation
   - Spray coverage visualization

## Notes

- Roboguide binary cache files and history SavePoints have been excluded from
  version control
- The simulation was built for a FANUC M-10iD/12 painting workcell
- Trajectory LS files generated by `trajectory/ply_ls_ALL_one.py` can be
  imported directly for validation

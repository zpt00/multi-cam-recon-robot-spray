# 项目背景与思路

## 项目简介

本项目构建"视觉感知 → 三维重建 → 轨迹规划 → 机器人执行"闭环系统。
使用多台 Intel RealSense D435 深度相机搭建多视角采集平台，实现工件稠密三维重建，
自动生成 FANUC 工业机器人喷涂轨迹程序并下发执行。

**项目周期**：2025年9月 — 2026年1月

**技术栈**：Python · Open3D · OpenCV · PyRealSense2 · NumPy/SciPy · FANUC TP/LS

## 主要工作

1. **多相机标定系统** — 基于 ChArUco 标定板的外参标定与最优筛选
2. **多视角点云融合** — 实时稠密三维重建，双路策略平衡精度与速度
3. **工件网格重建** — 点云后处理与曲面重建
4. **喷涂轨迹规划** — 自动生成六自由度喷枪轨迹，输出 FANUC LS 程序
5. **机器人集成部署** — 手眼标定、FTP 上传、PLC 协同，产线实际运行

## 成果

系统在 FANUC 工业机器人喷涂产线实际部署验证，实现从点云采集到机器人执行的全自动流程。

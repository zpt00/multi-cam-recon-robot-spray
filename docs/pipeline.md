# 系统技术概述

## 管线流程

```
标定 → 融合 → 后处理 → 配准 → 网格重建 → 轨迹规划 → 机器人通讯 → 仿真验证
```

## 各阶段概要

### 1. 多相机标定
采用 ChArUco 标定板进行四相机外参标定。以 cam0 为参考坐标系，通过 solvePnP 求解各相机位姿，再链式计算相机间变换关系。对多次采集结果用统计算法筛选最优外参。

### 2. 点云融合
将四路深度图转为点云，利用外参统一变换到参考坐标系。通过 ICP 微调修正残差，可选 TSDF 体积融合。采用双路策略平衡精度与实时性。

### 3. 后处理
融合点云经包围盒裁剪、平面分割、离群滤波得到干净工件点云。

### 4. 配准与网格重建
将扫描点云与 CAD 参考模型配准，通过曲面重建生成水密三角网格。

### 5. 轨迹规划
基于工件网格自动生成六自由度喷枪轨迹，包含姿态定向、平滑处理与速度规划。输出 FANUC LS 程序文件。

### 6. 机器人通讯与自动化
通过手眼标定将轨迹转换到机器人基坐标系，FTP 上传 LS 文件，UDP 与 PLC 握手实现产线循环作业。

## 技术栈

Python · Open3D · OpenCV · Intel RealSense SDK · NumPy/SciPy · FANUC TP/LS · FTP/UDP

## 部署验证

系统已在 FANUC M-10iD/12 工业机器人喷涂产线实际部署，标定精度平移 <5mm、旋转 <1°，喷涂覆盖率 >90%。

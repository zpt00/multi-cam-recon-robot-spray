# 系统技术概述

## 管线流程

```
标定 → 融合 → 后处理 → 配准 → 网格重建 → 轨迹规划 → 机器人通讯 → 仿真验证
```

## 各阶段概要

### 1. 多相机标定
确定多台相机之间的空间关系，以一台为参考建立统一坐标系。

### 2. 点云融合
将多路深度数据拼接为完整工件点云。

### 3. 后处理
对融合点云进行裁剪、分割、去噪，提取工件区域。

### 4. 配准与网格重建
将扫描数据与参考模型对齐，生成工件三维网格。

### 5. 轨迹规划
基于工件模型自动生成六自由度喷枪喷涂路径。

### 6. 机器人通讯与自动化
将轨迹转换到机器人坐标系，上传至控制器，配合 PLC 实现产线循环作业。

## 技术栈

Python · Open3D · OpenCV · Intel RealSense SDK · NumPy/SciPy · FANUC TP/LS · FTP/UDP

## 部署验证

系统已在 FANUC 工业机器人喷涂产线实际部署运行。
